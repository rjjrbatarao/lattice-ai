# About
Lattice AI is an experimental GPU sharing protocol with new Proof of Inference consensus mechanism built for validating AI inferences. Fully web based and uses WebGPU to share compute resources. Protect compute using x402 scheme. 

## Proof of work vs Proof of stake vs Proof of inference
PoW and PoS stake concensus mechanism were built with cryptographic difficulty to protect itself and make the whole cyptocurrency secure, on the other hand this cryptography has no other use than burning coal with no realwork compute work of just brute hashing. Proof of inference works out by using realworld inference to secure itself without wasting precious compute power, making it greener alternative to the latter mechanism since compute required for the system security has also realworld inference use.
